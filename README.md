# Experimental Evaluation of Planetary Rover Planning

This repository contains the experimental study developed for Research
Track II.

The study investigates how onboard memory capacity, scientific dataset
characteristics, and time-dependent data degradation affect planetary
rover planning performance.

The planning model is adapted from a Planetary Rover project previously
developed for the AI2 course. This repository extends that model with:

- automated problem generation;
- repeated experimental trials;
- ENHSP output parsing;
- quantitative metrics;
- statistical analysis;
- data visualization;
- a reproducible Jupyter notebook.

## Main research question

How do onboard memory constraints and time-dependent data degradation
affect the feasibility and efficiency of planetary rover data-collection
missions planned using PDDL+?
